## 🧠 About The Project

**Missionaries & Cannibals** is a brain-teasing logic puzzle brought to life with smooth animations, vibrant colors, and an intuitive UI — all in a single HTML file.  
Your goal? Safely ferry all missionaries and cannibals across the river without letting the cannibals outnumber missionaries on either side.  

Sounds easy? Think again. 😏

---

## 🔍 Features

- 🎮 Fully playable **classic logic puzzle**
- ✨ Smooth **CSS animations** for boat movement and characters
- 🌊 **Animated water effect** for realism
- 🧩 **Pure HTML, CSS, JavaScript** — no images or external assets
- 📱 **Responsive** — works on desktop & mobile
- 🧠 **Logical constraints** prevent invalid moves
- 🚀 All code in **one single HTML file**

## 🎯 How to Play

1. Start with 3 missionaries and 3 cannibals on one side.
2. Move the boat by selecting characters to travel.
3. Don’t let cannibals outnumber missionaries on either bank!
4. Successfully get everyone across to win. 🏆
